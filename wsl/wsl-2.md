
# WSL 2.0
## Install WSL 2 in Windows
PowerShell
```shell
wsl --install
```

Install Ubuntu or other system from Microsoft Store

## Access the remote OS from windows
* in the windows explorer addres type
```
WSL$
```

Access windows from the remote OS
```shell
cd /mnt
```
